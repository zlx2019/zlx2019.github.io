Rust【八】集合容器

发表于2023-03-24|更新于2023-04-16|Rust
|字数总计:769|阅读时长:3分钟|阅读量:

前言

Rust 标准库中包含一系列被称为 集合（collections）的非常有用的数据结构。大部分其他数据类型都代表一个特定的值，不过集合可以包含多个值。

Vector

Vec是Rust中提供的一种可自动扩容的集合容器。就类似于Java中的ArrayList一般。同样是基于数组所实现的。允许我们在一个单独的数据结构中储存多个值，所有值在内存中彼此相邻排列。Vec只能储存相同类型的值。并且Vec是在堆上分配的。

定义Vec

定义一个空的Vec

通过Vec的关联函数Vec::new,可以创建一个空的Vector:

let mut list : Vec<i32> = Vec::new();

定义一个有初始值的Vec

通过Vec![]宏来创建一个Vec容器,并且可以直接设置一些初始值。

let mut list: Vec<i32> = Vec![1,2,3];

添加元素

通过Vec的push()方法,可以向容器内尾部添加元素。

list.push(1);
list.push(2);
list.push(3);

println!("{:?}",list); //[1, 2, 3]

也可以通过索引来进行元素的修改,前提示修改的索引元素必须存在,否则就会panic。

list[0] = 11;
println!("{:?}",list); //[11, 2, 3]

获取元素

通过数组的索引获取

println!("{}",&list[2])//3

通过方法下标来获取

println!("{}",list.get(2).unwrap());//3

删除元素

list.remove(0); 
println!("{:?}",list);//[2, 3]

也可以通过clear()直接清空所有元素.

list.clear();
println!("{:?}",list);//[]

遍历元素

fn main() {
    let mut list: Vec<i32> = vec![1,2,3,4,5];
    for itme in &list  {
        println!("{}",itme)
    }
}

String

称作String的类型是由标准库提供的,而没有写进核心语言部分，它是可增长的、可变的、有所有权的、UTF-8 编码的字符串类型。当 Rustacean 们谈到 Rust 的 “字符串”时，它们通常指的是 String和字符串 slice &str 类型，而不仅仅是其中之一。虽然本部分内容大多是关于 String ，不过这两个类型在 Rust 标准库中都被广泛使用，String 和字符串 slice 都是 UTF-8 编码。

定义字符串的方式

新建一个空字符串

let s1: String = String::new();

通过字符串切片转换为字符串

let s2: String = "Hello".to_string();

基于字符串切片,创建一个字符串

let s3: String = String::from("Hello");

由于Rust中的字符串是基于UTF-8编码的,所以它可以支持任何国家的语言。

let hello = String::from("السلام عليكم");
let hello = String::from("Dobrý den");
let hello = String::from("Hello");
let hello = String::from("שָׁלוֹם");
let hello = String::from("नमस्ते");
let hello = String::from("こんにちは");
let hello = String::from("안녕하세요");
let hello = String::from("你好");
let hello = String::from("Olá");
let hello = String::from("Здравствуйте");
let hello = String::from("Hola");

拼接字符串

let mut text = String::from("hello");
text.push_str("_word");
text.push('~');
println!("{}",text) //hello_word~

通过+操作符来拼接

fn main() {
    let s1 = String::from("hello");
    let s2 = String::from("_wold!");
    let s3 = s1 + &s2; //注意这里, s1的所有权将会失效,因为转移到s3身上了
    println!("{}",s3);//hello_wold!
}

通过字符串格式化

fn main() {
    let s1 = String::from("hello");
    let s2 = String::from("_wold!");
    println!("{}{}",s1,s2);//hello_wold!
}
