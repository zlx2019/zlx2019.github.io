Centos7-ARM-Docker安装

发表于2023-05-29|更新于2023-05-29|Docker
|字数总计:1.1k|阅读时长:5分钟|阅读量:

本章主要讲述如何在Centos7-ARM64架构安装Docker。

1. 更改yum源国内镜像

  1. 备份原镜像,防止无法恢复

    mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.bak

  2. 下载yum源(阿里云镜像)

    wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo

  3. 清理并且更新包

    yum clean all
yum makecache
yum update

arm架构的yum源配置暂时未能成功通过阿里云镜像，使用网易镜像成功创建。

  1. 备份原镜像,以备无患

    mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.bak

  2. 创建以下三个文件

  3. 刷新yum缓存

    yum clean all & yum makecache

2. 安装Docker

1. 安装所需依赖

yum install -y yum-utils device-mapper-persistent-data lvm2

2. 添加Docker源

这里使用阿里云源,速度会比较快

sudo yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

3. 安装Docker

yum install docker-ce docker-ce-cli containerd.io

默认会安装最新版的,也可以安装指定版本的,如下:

# 列出可用的版本
yum list docker-ce --showduplicates | sort -r
# 安装指定版本的Docker
yum install -y docker-ce-18.09.6 docker-ce-cli-18.09.6 containerd.io

4. 启动与停止

启动Docker

sudo systemctl daemon-reload
sudo systemctl start docker

关闭Docker

sudo systemctl stop docker
或者
sudo systemctl stop docker.socket
sudo systemctl stop docker.service

设置Docker开机启动

systemctl enable docker

5. 镜像加速

国内Docker镜像仓库名称 链接
Docker 官方中国区 https://registry.docker-cn.com
网易 http://hub-mirror.c.163.com
中国科学技术大学 https://docker.mirrors.ustc.edu.cn
阿里云 https://<你的ID>.mirror.aliyuncs.com

修改/etc/docker/daemon.json文件,如果没有创建即可:

sudo vim /etc/docker/daemon.json

文件内容:

{
  "registry-mirrors": ["http://hub-mirror.c.163.com"]
}

重启Docker

sudo systemctl restart docker

然后就可以查看一下是否修改成功:

# docker info

....
 Registry Mirrors:
  http://hub-mirror.c.163.com
 Live Restore Enabled: false
