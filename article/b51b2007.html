Java【反射】

发表于2021-06-03|更新于2023-03-22|Java
|字数总计:1.4k|阅读时长:6分钟|阅读量:

前言

反射机制在程序运行时，对于任意一个类，都能够知道这个类的所有属性和方法；对于任意一个对象，都能够调用它的任意一个方法和属性。这种 动态的获取信息 以及 动态调用对象的方法 的功能称为 java 的反射机制,Java本身是静态语言,但是通过反射拥有了动态语言的特性。

基础类准备

/**
 * 学生类
 * @author lx Zhang.
 * @date 2021/6/3 1:15 下午
 */
public class Student implements Serializable {
    public Student() { }

    public Student(String name, Integer age, String address) {
        this.name = name;
        this.age = age;
        this.address = address;
    }

    public String name;
    public Integer age;
    public String address;
}

1. 传统方式

一般情况下，我们使用某个类时必定知道它是什么类，是用来做什么的。于是我们直接对这个类进行实例化，之后使用这个类对象进行操作。

Student student = new Student(); //直接初始化并且赋值
student.age = 19;

2. 反射方法

//通过反射方式,根据全类名进行创建其实例对象,<?>表示未知的类型
Class<?> cls = Class.forName("com.lxz.reflect.Student");
//创建实例
Object instance = cls.newInstance();

3.反射常用API

在反射中，要获取一个类或调用一个类的方法，我们首先需要获取到该类的 Class 对象。

在 Java API 中，获取 Class 类对象有三种方法：

  1. 通过反射 Class.forName静态方法

    Class<?> cls = Class.forName("com.lxz.reflect.Student");

  2. 数据类型.class

    Class<String> stringClass = String.class;

  3. 通过类对象的getClass()方法获取

    Class studentClass = student.getClass();

1.通过反射创建对象

第一种: 通过Class对象的newInstance()方法

//1.
Class<?> cls = Class.forName("com.lxz.reflect.Student");
Student instance = (Student) cls.newInstance();

//2.
Class<?> clss = Student.class;
Student instance1 = (Student) clss.newInstance();

第二种: 通过Constructor对象的newInstance()方法

//1. 通过Class的Constructor创建对象,这里使用的是无参构造
Class<?> cls = Class.forName("com.lxz.reflect.Student");
Constructor<?> constructor = cls.getConstructor();
Student instance = (Student) constructor.newInstance();

2.通过反射获取构造函数

获取无参构造函数

Class<?> cls = Class.forName("com.lxz.reflect.Student");
Constructor<?> constructor = cls.getConstructor();
Student instance = (Student) constructor.newInstance();

获取全参构造函数

//1. 通过Class的Constructor创建对象
Class<?> cls = Class.forName("com.lxz.reflect.Student");
//2.根据带参函数的参数列表 指定类型
Constructor<?> constructor = cls.getConstructor(String.class,Integer.class,String.class);
Student instance = (Student) constructor.newInstance("小明",18,"北京");

3.通过反射获取属性和值

通过反射默认情况无法获取private修饰的属性

//在获取值之前设置
field.setAccessible(true);

调用Field.setAccessible(true)的意思是，别管这个字段是不是public，一律允许访问。

//1. 通过Class的Constructor创建对象
Class<?> cls = Class.forName("com.lxz.reflect.Student");
Student instance = (Student) cls.newInstance();
//2.获取属性Class
Field field = cls.getField("name");
//3.通过属性获取对象的值
String name = (String) field.get(instance);

4.反射获取所有属性和值

cls.getFields();方法可以获取 Class 类的属性，但无法获取私有属性。

而如果使用 Class 对象的 getDeclaredFields() 方法则可以获取包括私有属性在内的所有属性

Class<?> cls = Class.forName("com.lxz.reflect.Student");
Constructor<?> constructor = cls.getConstructor(String.class, Integer.class, String.class);
Student instance = (Student) constructor.newInstance("小张", 18, "河北");

Field[] fields = cls.getFields();
for (Field f : fields) {
  System.out.println(f + "--->" + f.get(instance));
}

结果如下

public java.lang.String com.lxz.reflect.Student.name--->小张
public java.lang.Integer com.lxz.reflect.Student.age--->18
public java.lang.String com.lxz.reflect.Student.address--->河北

5. 通过反射设置属性值

Class<?> cls = Class.forName("com.lxz.reflect.Student");
Object o = cls.newInstance();

//获取name属性,并且设置Value
Field field = cls.getField("name");
field.set(o,"小明");

6.通过反射调用方法

1. 调用无参数方法

public void sayHello(){
  System.out.println("hi i name is " + this.name);
}
//1. 通过Class的Constructor创建对象
Class<?> cls = Class.forName("com.lxz.reflect.Student");
//2. 获取对象实例
Object o = cls.newInstance();
//3.获取方法Class对象,没有参数
Method method = cls.getMethod("sayHello");
//4.执行获取到的method
method.invoke(o);

2. 调用带有参数方法

public void sayHi(String name){
  System.out.println("hi i name is" + name);
}
//1. 通过Class的Constructor创建对象
Class<?> cls = Class.forName("com.lxz.reflect.Student");
//2. 获取对象实例
Object o = cls.newInstance();
//3.获取方法Class对象,指定参数类型
Method method = cls.getMethod("sayHi", String.class);
//4.执行获取到的method,携带参数
method.invoke(o,"法外狂徒张三");

3. 调用带参数带返回值的方法

public String appendName(String str1,String str2){
  return str1 + str2;
}
//1. 通过Class的Constructor创建对象
Class<?> cls = Class.forName("com.lxz.reflect.Student");
//2. 获取对象实例
Object o = cls.newInstance();
//3.获取方法Class对象,指定参数类型
Method method = cls.getMethod("appendName", String.class, String.class);
//4.执行获取到的method,携带参数
String result = (String) method.invoke(o,"hello","word");

4. 调用静态方法

如果获取到的Method表示一个静态方法，调用静态方法时，由于无需指定实例对象，所以invoke方法传入的第一个参数永远为null

public static void show(Integer age){
  System.out.println("i now age is " + age);
}
//1. 通过Class的Constructor创建对象
Class<?> cls = Class.forName("com.lxz.reflect.Student");
//2. 获取对象实例
Object o = cls.newInstance();
//3.获取方法Class对象,指定参数类型
Method method = cls.getMethod("show", Integer.class);
//4.执行获取到的method,携带参数
method.invoke(null,18);

7. 通过反射获取继承关系

1. 获取父类Class

Class<?> cls = Class.forName("com.lxz.reflect.Student");
Class<?> superclass = cls.getSuperclass();

2. 获取实现的接口

Class<?> cls = Class.forName("com.lxz.reflect.Student");
Class<?>[] interfaces = cls.getInterfaces();
for (Class<?> item : interfaces) {
  System.out.println(item.getName());
}

3. 获取注解

Class<?> cls = Class.forName("com.lxz.reflect.Student");
Annotation[] annotations = cls.getAnnotations();
for (Annotation annotation : annotations) {
  System.out.println(annotation);
}
