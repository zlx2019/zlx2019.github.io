Go核心篇章-reflect

发表于2023-05-02|更新于2023-05-26|Go
|字数总计:2.9k|阅读时长:11分钟|阅读量:

什么是反射?

反射(reflect)是静态语言的一种特性,它允许程序在运行时对一个值进行一系列操作,如获取值、修改值、调用方法。这种机制被称之为反射(reflect)。
反射是Go语言中基础中的基础,也是核心中的核心,通过反射我们可以做到一些只有类似于动态语言才能做到的事,包括框架底层设计、扩展性、通用性。

为什么需要反射?

反射是一种非常强大的特性，它能够在运行时动态地获取和操作程序本身的信息，包括变量、函数、类型等。在Go中，反射的应用场景非常广泛，其中一些主要的应用场景包括:

  1. 类型判定

    例如当我们对一个未知类型的变量进行统一操作时,这个时候就需要通过反射来断言该未知类型的变量究竟是什么类型，这样才能更好的针对性进行处理操作。

  2. 依赖注入和插件系统

    在编写框架、组件、插件等应用程序时，常常需要使用依赖注入和插件系统来实现可扩展性和灵活性。反射可以帮助程序在运行时动态地加载和管理插件，或者实现依赖注入的功能。

  3. 动态操作

    像我们平常开发过程中,对一个值进行操作,这种一般都是静态操作,因为类型已经确定,所以可以操作的空间是有限的,而通过反射对一个值进行操作则被称之为动态操作,它的局限性非常大。

反射虽然非常强大，但同时也带来了一些问题。比如，使用反射会影响程序的性能，而且反射代码通常比较复杂，可读性差。因此，在使用反射时需要慎重考虑其适用性和效率，以确保代码的可读性和性能。

核心类型与函数

Go的反射是标准库中的reflect包提供的,它提供了两个非常重要的核心类型:reflect.Type、reflect.Value。可以理解为所有的值在反射中都由这两个类型所组成。

并且该包还提供了两个重要函数:reflect.TypeOf()、reflect.ValueOf(),这两个函数分别用于获取一个值的Type和Value。

  • Type

    Type表示一个Go类型,它是一个接口。它持有类型相关的元信息: 如类型名称、大小、类型本身方法等。

  • Value

    Value表示值的元信息。比如值的类型、实际值、是否可修改等。

如下示例:

func main() {
	var name string = "张三"
	typeInfo := reflect.TypeOf(name)
	valueInfo := reflect.ValueOf(name)
	fmt.Println(typeInfo)  //string
	fmt.Println(valueInfo) //张三
}

从以上执行结果就可以看出:

name的Type就是string。

name的Value就是张三这个值。

Type

reflect.Type表示一个Go的值类型。具体来说它是一个接口,它持有类型的所有信息。

reflect.Type通常是用reflect.TypeOf()所获取。

源码如下:

type Type interface {
	Align() int
	FieldAlign() int
	Method(int) Method
	MethodByName(string) (Method, bool)
	NumMethod() int
	Name() string
	PkgPath() string
	Size() uintptr
	String() string
	Kind() Kind
	Implements(u Type) bool
	AssignableTo(u Type) bool
	ConvertibleTo(u Type) bool
	Comparable() bool
	Bits() int
	ChanDir() ChanDir
	IsVariadic() bool
	Elem() Type
	Field(i int) StructField
	FieldByIndex(index []int) StructField
	FieldByName(name string) (StructField, bool)
	FieldByNameFunc(match func(string) bool) (StructField, bool)
	In(i int) Type
	Key() Type
	Len() int
	NumField() int
	NumIn() int
	NumOut() int
	Out(i int) Type
	common() *rtype
	uncommon() *uncommonType
}

示例如下:

func main() {
	var i int = 3
	//获取i的Type
	typeInfo := reflect.TypeOf(i)

	// i的类型
	fmt.Println(typeInfo) //int

	// 获取Type的类型名称
	fmt.Println(typeInfo.Name()) //int

	// 获取Type的原始类型
	fmt.Println(typeInfo.Kind()) //int
	fmt.Println(typeInfo.Kind() == reflect.Int) // true

	// 获取Type所占字节大小
	fmt.Println(typeInfo.Size()) // 8
}

Kind()

Go中的每一个类型在反射中都有一个原始类型,如int类型通过反射获取到的原始类型就是reflect.Int类型。

通过Type.Kind()获取到的就是一个类型的原始类型。

Go反射中的所有原始类型有如下这些:

通常我们会经常使用switch语句,来对一个不确定的Kind原始类型来进行类型断言。如下所示

示例: 断言类型

  1. 非反射方式

    var i interface{} = 3
switch i.(type) {
  case int:
case uint:
case string:
case map[string]any:

case ...:
default:
}

  2. 反射原始类型方式

    通过Type的Kind原始类型来进行断言,可以做到更多的类型匹配。

    var i interface{} = 3
switch reflect.TypeOf(i).Kind() {
case reflect.Int:
case reflect.Uint:
case reflect.String:
case reflect.Slice:
case reflect.Map:
case reflect...:

default:
  ....
}

Elem()

Type的Elem()方法用于获取元素值的Type。该方法只用于Array、Chan、Map、Pointer、Slice等类型。其他类型调用后会直接panic。

func main() {
	list := []int{1,2,3,4}
	types := reflect.TypeOf(list)
	fmt.Println(types.Kind() == reflect.Slice) //true
	fmt.Println(types.Elem()) // int
	fmt.Println(types.Elem().Kind() == reflect.Int) //true
}

Size()

Size()用于获取Type所占字节大小。

func main() {
	fmt.Println(reflect.TypeOf((int8(1))).Size())  //1
	fmt.Println(reflect.TypeOf((int16(1))).Size()) //2
	fmt.Println(reflect.TypeOf((int32(1))).Size()) //4
	fmt.Println(reflect.TypeOf((int64(1))).Size()) //8
	fmt.Println(reflect.TypeOf("").Size())    //16
}

为了简化,可以直接使用unsafe.Sizeof()来获取一个值的字节所占大小。

fmt.Println(unsafe.Sizeof(1)) //8

Value

reflect.Value表示一个具体的值,它持有值的所有信息。

有如下一个结构体:

type Model struct {

}

func (*Model) Say()  {
	fmt.Println("Say")
}

func (*Model) Hello()  {
	fmt.Println("Hello")
}

示例: 获取Model的值信息

func main() {
	m := &Model{}
	// 获取m的值信息
	value := reflect.ValueOf(m)
	// 获取动态值的类型信息,也就是传入ValueOf方法的变量类型
	fmt.Println(value.Type()) // *main.Model
	fmt.Println(value.Kind()) // ptr
	fmt.Println(value.NumMethod())//2
}

Type()

Value的Type()方法是用于获取值的类型信息的方法。

// 获取一个值的具体类型
fmt.Println(reflect.ValueOf(1).Type()) // int
fmt.Println(reflect.ValueOf(Model{}).Type()) // Model
fmt.Println(reflect.ValueOf(&Model{}).Type())// *Model

Kind()

与Type.Kind()方法一致,获取的是值的原始类型。

fmt.Println(reflect.ValueOf(1).Kind()) // int
fmt.Println(reflect.ValueOf(Model{}).Kind()) // struct
fmt.Println(reflect.ValueOf(&Model{}).Kind())// ptr

Interface()

Interface()方法用于将一个Value还原成一个any空接口对象。

func main() {
	i := reflect.ValueOf(&Model{}).Interface()
	switch i.(type) {
	case *Model:
		fmt.Println("是*Model类型") //是*Model类型
	}
}

Elem()

Value的Elem()方法和Type的Elem()方法不同。它在这里是用来获取Value具体的值。如果是指针类型则返回指针所指向的具体值。

如果值不是接口或者指针类型,则会产生panic。

如果指针类型为nil,则返回默认值。

type Model struct {
	No int
	Age int
}

func main() {
	fmt.Println(reflect.ValueOf(&Model{}).Elem()) //{0 0}
	fmt.Println(reflect.ValueOf(&Model{10,20}).Elem()) //{10 20}
  fmt.Println(reflect.ValueOf(Model{10,20}).Elem()) //panic
}

Field相关

NumField()

NumField()方法用于获取值所拥有的字段数量。值如果不是struct则会导致panic。

type Model struct {
	No int
	Age int
}
func main() {
	val := Model{No: 10,Age: 19}
	value := reflect.ValueOf(val)
	fmt.Println(value.NumField()) //2
}

Field(int)

获取结构体的第n个字段Value。

func main() {
	val := Model{No: 10,Age: 19}
	value := reflect.ValueOf(val)

	// 获取第一个字段
	field := value.Field(0)
	fmt.Println(field)// 10
	// 获取第二个字段
	field = value.Field(1)
	fmt.Println(field) //19
}

FieldByName(string)

根据字段名,获取字段的Value值。

func main() {
	val := Model{No: 10,Age: 19}
	value := reflect.ValueOf(val)

	// 获取第一个字段
	field := value.FieldByName("No")
	fmt.Println(field)// 10
	// 获取第二个字段
	field = value.FieldByName("Age")
	fmt.Println(field) //19
}

获取Tag标签

type Model struct {
	No int  `coustomTag:"nos"`
	Age int
}

func main() {
	types := reflect.TypeOf(&Model{}).Elem()
	field, _ := types.FieldByName("No")
	fmt.Println(field.Tag.Get("coustomTag")) // nos
}

Method相关

NumMethod()

NumMethod用于获取一个值下所持有的方法数量.

NumMethod 返回值的方法集中的方法数。对于非接口类型，它返回导出方法的数量。对于接口类型，它返回导出和未导出方法的数量。

type Model struct {}
func (*Model) Say()  {}
func (*Model) Hello()  {}

func main() {
	fmt.Println(reflect.TypeOf(&Model{}).NumMethod()) // 2
}

Method(int)

Method(int)根据索引获取Value的第n个方法。

func main() {
	val := &Model{No: 10,Age: 19}
	value := reflect.ValueOf(val)

	// 获取值的第一个方法
	method := value.Method(0)
	fmt.Println(method.Kind()) // func
	fmt.Println(method.Kind() == reflect.Func) //true
}

MethodByName(string)

MethodByName(string)根据指定的方法名,获取Value下的方法。

// 根据方法名 获取Value的方法
method = value.MethodByName("Hello")
fmt.Println(method.Kind()) // func
fmt.Println(method.Kind() == reflect.Func) //true

方法如果不存在,则返回一个(Invalid)无效的类型。

// 根据方法名 获取Value的方法
method = value.MethodByName("Hellosss")
fmt.Println(method.Kind()) // invalid
fmt.Println(method.Kind() == reflect.Invalid) //true

Call([]Value)

Value的Call()方法用来执行Type为Func的Value。

简单来说就是用来执行方法的。

示例一: 执行无参数方法。

func (*Model) Hello()  {
	fmt.Println("Hello Func")
}

func main() {
	val := &Model{No: 10,Age: 19}
	value := reflect.ValueOf(val)
	// 根据方法名 获取Value的方法
	method := value.MethodByName("Hello")
	// 方法是否存在(有效)
	if method.IsValid() {
		// 执行无参数方法
		method.Call([]reflect.Value{}) //Hello Func
	}
}

示例二: 执行有参数方法。

func (*Model) Params(name string, age int) {
	fmt.Printf("Name: %s Age: %d",name,age)
}

func main() {
	val := &Model{No: 10,Age: 19}
	value := reflect.ValueOf(val)
	// 根据方法名 获取Value的方法
	method := value.MethodByName("Params")
	// 方法是否存在(有效)
	if method.IsValid() {
		// 构建方法所需参数
		args := []reflect.Value{reflect.ValueOf("张三"),reflect.ValueOf(18)}
		// 执行有参数方法
		 method.Call(args) // Name: 张三 Age: 18
	}
}

示例三: 执行有参数有返回值方法。

func (*Model) Params(name string, age int) string {
	return fmt.Sprintf("i name is %v age is %v",name,age)
}

func main() {
	val := &Model{No: 10,Age: 19}
	value := reflect.ValueOf(val)
	// 根据方法名 获取Value的方法
	method := value.MethodByName("Params")
	// 方法是否存在(有效)
	if method.IsValid() {
		// 构建方法所需参数
		args := []reflect.Value{reflect.ValueOf("张三"),reflect.ValueOf(18)}
		// 执行有参数方法,获取返回值
		results := method.Call(args)
		// 输出返回值
		fmt.Println(results[0].Interface()) //i name is 张三 age is 18
	}
}

使用反射操作变量

每一个变量、或者结构体的字段都被称之为Field,也就是一个Value。我们可以通过Set(Value)一系列方法来修改这个变量的值。如下两个案例:

操作一个指针对象

func main() {
	val := &Model{No: 10,Age: 19}
	// 获取指针所指向的具体值
	value := reflect.ValueOf(val).Elem()
	// 根据字段名获取字段
	field := value.FieldByName("No")
	// 设置新的数据
	field.SetInt(100)
  // 或者这样更新
  field.Set(reflect.ValueOf(100))
	fmt.Println(val.No) //100
  fmt.Println(val) //{100 19}
}

操作非指针对象

这种方式有些过于复杂了,建议使用指针方式。

func main() {
	val := Model{No: 10,Age: 19}
	// 获取Value
	value := reflect.ValueOf(val)
	// 创建Value对应的具体值对象
	elem := reflect.New(value.Type()).Elem()
	// 将原来的值拷贝给新的值对象
	elem.Set(value)
	// 获取新的值对象No字段
	field := elem.FieldByName("No")
	// 修改No字段
	field.SetInt(100)
	// 类型强转
	newModel := elem.Interface().(Model)
	fmt.Println(newModel) //{100 19}
}

使用反射创建实例

使用reflect.New(Type)方法可以创建一个新的Value。入参为一个Type类型。

// 通过反射创建一个string类型的Value
str := reflect.New(reflect.TypeOf("")).Elem()
// 为Value设置值
str.SetString("Hello")
// 判断原始类型
fmt.Println(str.Kind() == reflect.String) //true
// 输出
fmt.Println(str) // Hello

示例: 通过一个Student实例,使用反射创建一个新的实例

type Student struct {
	Name string
}

func Build(obj any) any {
	// 获取到值的类型
	objType := reflect.TypeOf(obj)
  // 如果时指针,就获取指针具体的值的类型
	if objType.Kind() == reflect.Ptr {
		objType = objType.Elem()
	}
	// 通过类型创建一个新的实例,并且获取到值
	newObj := reflect.New(objType).Elem()
	// 获取新值的Name字段
	name := newObj.FieldByName("Name")
	// 为Name字段设置值
	name.SetString("张三")
	return newObj.Interface()
}

func main() {
	s := Student{}
	news := Build(s)
	fmt.Println(news.(Student).Name) // 张三
}
