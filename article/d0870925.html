关于我的Debian系统配置

设置允许root用户ssh登录

在/etc/ssh/sshd_config文件中添加如下配置:

PermitRootLogin yes

重启ssh服务

$ service ssh restart

Debian镜像下载

官方镜像

请根据自己的需要下载不同的架构镜像，这里推荐离线安装方式，所以下载DVD格式镜像;

具体安装细节请参考录制的视频;

加入sudo用户组

# 切换到root用户
su -

# 普通用户加入到sodu用户组
usermod -aG sudo [普通用户名]

#推荐加入dialout用户组使用串口更方便
usermod -a -G dialout 要加入的用户名

# 退出root用户
exit

apt国内镜像源

修改/etc/apt/sources.list配置文件,设置为清华源：

# 默认注释了源码镜像以提高 apt update 速度，如有需要可自行取消注释
deb https://mirrors.tuna.tsinghua.edu.cn/debian/ bookworm main contrib non-free non-free-firmware
# deb-src https://mirrors.tuna.tsinghua.edu.cn/debian/ bookworm main contrib non-free non-free-firmware

deb https://mirrors.tuna.tsinghua.edu.cn/debian/ bookworm-updates main contrib non-free non-free-firmware
# deb-src https://mirrors.tuna.tsinghua.edu.cn/debian/ bookworm-updates main contrib non-free non-free-firmware

deb https://mirrors.tuna.tsinghua.edu.cn/debian/ bookworm-backports main contrib non-free non-free-firmware
# deb-src https://mirrors.tuna.tsinghua.edu.cn/debian/ bookworm-backports main contrib non-free non-free-firmware

# deb https://mirrors.tuna.tsinghua.edu.cn/debian-security bookworm-security main contrib non-free non-free-firmware
# # deb-src https://mirrors.tuna.tsinghua.edu.cn/debian-security bookworm-security main contrib non-free non-free-firmware

deb https://security.debian.org/debian-security bookworm-security main contrib non-free non-free-firmware
# deb-src https://security.debian.org/debian-security bookworm-security main contrib non-free non-free-firmware

安装 ohmyzsh

1.安装git

$ sudo apt install git -y

2.安装zsh

$ sudo apt install zsh -y
$ chsh -s /bin/zsh

2.安装ohmyzsh

# 下载 ohmyzsh
$ git clone https://gitee.com/oldsyang/ohmyzsh.git ~/.oh-my-zsh

# 拷贝默认配置模板
$ cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc

# 刷新zsh
$ chsh -s /bin/zsh

4.安装主题

编辑~/.zshrc配置文件，修改ZSH_THEME属性:

ZSH_THEME="powerlevel10k/powerlevel10k"

下载主题

$ git clone --depth=1 https://gitee.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

接下来就可以自定义样式了，

5.安装字体

以下面的字体为例:

  • MesloLGS NF Regular.ttf
  • MesloLGS NF Bold.ttf
  • MesloLGS NF Italic.ttf
  • MesloLGS NF Bold Italic.ttf

创建字体目录

$ sudo mkdir /usr/share/fonts/MesloLgsNF

将下载好的ttf字体文件拷贝到该目录，然后依次执行如下命令:

$ sudo mkfontscale
$ sudo mkfontdir
$ sudo fc-cache

此时文件夹会多两个文件：fonts.dir 以及 fonts.scale，安装成功.
