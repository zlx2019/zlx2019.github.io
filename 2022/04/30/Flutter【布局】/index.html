前言

页面布局是非常非常重要的一部分,一个优美且简介的布局会大大提升用户的体验感,美好的事物谁会不喜欢呢?本章主要介绍目前移动端比较常用也比较常见的布局。

1. 布局的介绍

线性布局

这种从上至下的布局几乎是目前移动端最常见的布局方式,被称为线性布局。

流式布局

而这种横向扩展的布局则被称为流式布局,因为当右侧的空间不足使用时,会自动向下流向,最终还是以垂直滑动。

弹性布局(Flex)

弹性布局也被称为Flex布局, 它于流式布局有些类似,不同的是它是基于比例的排布,如三个部件,在同一层次分别以1:2:3的形式所排布。

层叠布局

2. 弹性布局

弹性布局也就是Flex布局,该布局会使用到Flex部件,通过它来实现。下面我们先来看一下Flex部件的属性

Flex({
    Key? key,
    required this.direction, //Flex 排列方向:垂直|水平
    this.mainAxisAlignment = MainAxisAlignment.start,//主轴对齐方式
    this.mainAxisSize = MainAxisSize.max,
    this.crossAxisAlignment = CrossAxisAlignment.center,//交叉轴(副轴)对齐方式。
    this.textDirection,
    this.verticalDirection = VerticalDirection.down, //子部件排列顺序:默认从上到下
    this.textBaseline, // NO DEFAULT: we don't know what the text's baseline should be
    this.clipBehavior = Clip.none,
    List<Widget> children = const <Widget>[],
  })

其中direction参数是必填项,因为该属性表示了Flex的布局方向是垂直还是水平。

direction: Axis.vertical //垂直
direction: Axis.horizontal//水平

children则就是该布局下的所有子部件集合。这里我们使用三个Container来演示,分别设置不同背景颜色和宽高各100。

import 'package:flutter/material.dart';
void main(){
  runApp(const LayoutFlex());
}
/// Flutter Flex弹性布局
class LayoutFlex extends StatelessWidget{
  const LayoutFlex({Key? key}) : super(key: key);
  @override
  Widget build(BuildContext context) {
    return  MaterialApp(
      title: "弹性布局",
      home: Flex(
        //排列方向: 垂直|水平
        direction: Axis.vertical,//垂直
        //子部件集合
        children: [
          Container(color: Colors.red,width: 100,height: 100,),
          Container(color: Colors.amberAccent,width: 100,height: 100,),
          Container(color: Colors.blue,width: 100,height: 100,)
        ],
      )
    );
  }
}

1. 布局方向

我们来看一下Flex布局垂直排列与水平排列的效果。

home: Flex(
  //排列方向
  direction: Axis.vertical,//垂直/纵向布局
  direction: Axis.horizontal,//水平/横向布局
)

2. 主轴排列

Flex的mainAxisAlignment参数: 主轴排列,子组件在容器内主轴的排列规则,有如下几种方式

enum MainAxisAlignment {
  /// 主轴起点位置
  start,
  /// 主轴末端位置
  end,
  /// 主轴中心位置
  center,
  ///首个部件与末尾部件分别紧挨着主轴的起点与末端位置,然后中间的部件则均匀排布开来
  spaceBetween,
  /// 很难解释,直接看图把
  spaceAround,
  ///按照主轴的排列方向, 将子部件均匀排布开来
  spaceEvenly,
}

接下来我们来看一下垂直方向的start、center、end选型是如何排列。

home: Flex(
  //排列方向: 垂直
  direction: Axis.vertical,
  mainAxisAlignment: start,//start、center、end
  //子部件集合
  children: ...
)

这是三种较为简单的主轴对齐方式,接下里看一下另外三种较为复杂的。

home: Flex(
  //排列方向: 垂直|水平
  direction: Axis.vertical,
  mainAxisAlignment: MainAxisAlignment.spaceBetween,//spaceBetween、spaceAround、spaceEvenly
  //子部件集合
  children: ...
)

3. 副轴排列

crossAxisAlignment属性表示副轴的排列方式,有如下可选

enum CrossAxisAlignment {
  ///与副轴起端对齐
  start,
  ///与副轴末端对齐
  end,
  //居中对齐
  center,
  //铺满副轴
  stretch,
  baseline,
}

接下来演示一下副轴的几种排列对齐方式,还是以 Flex垂直布局、主轴排列居中为例。

home: Flex(
  direction: Axis.vertical,//纵向布局
  mainAxisAlignment: MainAxisAlignment.center,//主轴居中对齐
  crossAxisAlignment: CrossAxisAlignment.start, //start、center、end、stretch 依次演示
  //子部件集合
  children: ...
)

verticalDirection属性为垂直排列顺序,默认是从上到下down，只有 纵向布局时才生效。

enum VerticalDirection {
  // 从下到上排列
  up,
	// 从上到下排列子部件
  down,
}

4. Flex比例布局

如何将Flex中的所有子部件按照比例去布局呢,让所有子部件铺满整个页面。这时候需要用到Expanded部件,通过它再结合Flex就能实现占比布局，使用比例布局后,主轴对齐方式不再生效。如下所示

import 'package:flutter/material.dart';
void main(){
  runApp(const LayoutFlex());
}
/// Flutter Flex弹性布局
class LayoutFlex extends StatelessWidget{
  const LayoutFlex({Key? key}) : super(key: key);
  @override
  Widget build(BuildContext context) {
    return  MaterialApp(
      title: "弹性布局",
      home: Flex(
        direction: Axis.vertical,
        crossAxisAlignment: CrossAxisAlignment.stretch, //交叉轴对齐方式
        //子部件集合,子部件都使用Expanded包裹,实现Flex 1:1:1布局
        children: [
          Expanded(flex: 1, child: Container(color: Colors.red,width: 100,height: 100,)),
          Expanded(flex: 1,child: Container(color: Colors.yellow,width: 100,height: 100,)),
          Expanded(flex: 1,child: Container(color: Colors.blue,width: 100,height: 100,))
        ],
      )
    );
  }
}

3. 线性布局

线性布局会使用到两个基部件 Row、 Column,这两个部件其实都继承 Flex。其实就是根据Flex的布局方向所决定,Row就是对应的Flex横向布局,而Column则对应Flex的纵向布局。所以线性布局就不再叙述,请直接参考弹性布局。

4. 流式布局

流式布局需要使用Wrap部件。老规矩,先看下它的属性

Wrap({
  Key? key,
  this.direction = Axis.horizontal, //主轴布局方向,与Flex的该属性一个意思
  this.alignment = WrapAlignment.start,//主轴对齐方式
  this.spacing = 0.0,//主轴间距
  this.runAlignment = WrapAlignment.start,//空间不足换行后的对齐方式
  this.runSpacing = 0.0, //层级间距
  this.crossAxisAlignment = WrapCrossAlignment.start,
  this.textDirection,//子组件在主轴方向的布局顺序
  this.verticalDirection = VerticalDirection.down,
  this.clipBehavior = Clip.none,
  List<Widget> children = const <Widget>[],//子部件集合
}) : assert(clipBehavior != null), super(key: key, children: children);

先创建好一个流式布局架子,然后我们一点一点演示。老样子,我们使用三个盒子模型来演示。

import 'package:flutter/material.dart';
void main(){
  runApp(const LayoutStream());
}
class LayoutStream extends StatelessWidget{
  const LayoutStream({Key? key}) : super(key: key);
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: "流式布局",
      home: Wrap(
        children: [
          Container(width: 100, height: 100, color: Colors.red,),
          Container(width: 100, height: 100, color: Colors.yellow,),
          Container(width: 100, height: 100, color: Colors.blue,),
        ],
      ),
    );
  }
}

1. 布局方向

direction属性: 与Flex的direction一致,可选纵向与横向。

home: Wrap(
  direction: Axis.horizontal,//水平,默认为水平
  direction: Axis.horizontal,//垂直
),

一般流式布局,都是水平布局,生活中常用的各大APP都是以这种布局,相信大家并不陌生,所以就不多演示。

5. 网格布局

网格布局是集成了以上所学习到的布局和特性的一种成品布局,通常使用起来更方便更有效率。

如下所示:

网格布局由GridView组件实现。

/// 网格布局
class LayoutGrid extends StatelessWidget{
  const LayoutGrid({Key? key}): super(key: key);
  @override
  Widget build(BuildContext context) {
    ///创建十张图片组件
    var list = <Widget>[];
    for(int i = 0;i <= 10; i++){
      list.add(SizedBox(
        height: 100,
        width: 100,
        child: Image.network(
          "https://music9501.oss-cn-beijing.aliyuncs.com/2022-05-04/3674fd8ac6224dbc8c3a887a2c190e4b.jpg",
          fit: BoxFit.fill,
        ),
      ));
    }
    return MaterialApp(
      title: "网格布局",
      home: Scaffold(
        appBar: AppBar(title: const Text("网格布局"),),
        body: Center(
          //使用 GridView组件,实现网格布局
          child: GridView(
            padding: EdgeInsets.all(30),//
            gridDelegate: const SliverGridDelegateWithFixedCrossAxisCount(
                crossAxisCount: 2, //布局内横轴渲染组件数量
                mainAxisSpacing: 8, //主轴层次间距
              crossAxisSpacing: 8   //副轴间距
            ),
            //布局内的子组件
            children: list,
          ),
        ),
      ),
    );
  }
}

GridView组件提供了更简洁的用法.

/// 网格布局
class LayoutGrid extends StatelessWidget{
  const LayoutGrid({Key? key}): super(key: key);
  @override
  Widget build(BuildContext context) {
    ///创建十张图片组件
    return MaterialApp(
      title: "网格布局",
      home: Scaffold(
        appBar: AppBar(title: const Text("网格布局"),),
        body: Center(
          //使用 GridView组件,实现网格布局
          child: GridView.builder(gridDelegate: const SliverGridDelegateWithFixedCrossAxisCount(
              crossAxisCount: 2,
              mainAxisSpacing: 8,
              crossAxisSpacing: 8
          ),
              padding: const EdgeInsets.all(30),
              itemCount: 10,//布局内渲染的组件数量
              itemBuilder: (context,index){
            return Image.network(
              "https://music9501.oss-cn-beijing.aliyuncs.com/2022-05-04/3674fd8ac6224dbc8c3a887a2c190e4b.jpg",
              fit: BoxFit.fill,
            );
          }),
        ),
      ),
    );
  }
}

6. PageView

PageView通常用来做页面轮播。

class PageViewExam extends StatelessWidget{
  const PageViewExam({Key? key}):super(key: key);
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: "PageView",
      home: Scaffold(
        appBar: AppBar(title: const Text("轮播图"),),
        body: SizedBox(
          height: 300,
          child: PageView(
            children: [
              Container(color: Colors.red,),
              Container(color: Colors.blue,),
              Container( color: Colors.yellow,)
            ],
          ),
        )
      ),
    );
  }
}

默认是横向滑动轮播。也可以纵向轮播。

class PageViewExam extends StatelessWidget{
  const PageViewExam({Key? key}):super(key: key);
  @override
  Widget build(BuildContext context) {
    //创建一个PageView 控制器
    PageController controller = PageController(
      initialPage: 0,//表示默认显示PageView.children中的第一个组件。
      keepPage: true
    );
    return MaterialApp(
      title: "PageView",
      home: Scaffold(
        appBar: AppBar(title: const Text("轮播图"),),
        body: SizedBox(
          height: 300,
          child: PageView(
            scrollDirection: Axis.vertical,//滚动方向: 纵向
            controller: controller,
            children: [
              Container(color: Colors.red,),
              Container(color: Colors.blue,),
              Container( color: Colors.yellow,)
            ],
          ),
        )
      ),
    );
  }
}

PageView常用的方式

class PageViewExam extends StatelessWidget{
  const PageViewExam({Key? key}):super(key: key);
  @override
  Widget build(BuildContext context) {
    //创建一个PageView 控制器
    PageController controller = PageController(
      initialPage: 0,//表示默认显示PageView.children中的第一个组件。
      keepPage: true
    );
    return MaterialApp(
      title: "PageView",
      home: Scaffold(
        appBar: AppBar(title: const Text("轮播图"),),
        body: PageView.builder(
            scrollDirection: Axis.vertical,//轮播的方向 垂直
            itemCount: 10,//itemBuilder 渲染的组件数量
            itemBuilder: (context,index){
              return Container(
                color: Colors.red,
                child: Center(
                  child: Text('$index',textScaleFactor: 5,),
                ),
              );
            }),
      ),
    );
  }
}
