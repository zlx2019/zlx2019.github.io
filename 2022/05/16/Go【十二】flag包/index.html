Go【十二】flag包

发表于2022-05-16|更新于2023-03-01|Go
|字数总计:630|阅读时长:2分钟|阅读量:

前言

Go语言内置了flag包,实现了对命令行参数的解析,一般用于传输项目所需配置参数。

1. 获取所有参数

通过os.Args,可以获得命令行所有的参数.

func main() {
	var args []string = os.Args
	for i, arg := range args {
		fmt.Printf("args[%d]=%s\n", i, arg)
	}
}

然后打包运行

go build main.go
./main a b c d

输出结果

args[0]=./main
args[1]=a
args[2]=b
args[3]=c
args[4]=d

这种方式稍微复杂,使用起来并不友好。

2. flag基本使用

1. 根据参数名和类型获取

flag包提供了很多根据不同数据类型,获取参数的函数,如flag.String()。

//格式
//flag.String(参数名称, 参数默认值, 参数描述)
name := flag.String("name", "张三", "姓名")
age := flag.Int("age", 18, "年龄")
sex := flag.Bool("sex", true, "性别")
fmt.Println("name:", *name)
fmt.Println("age:", *age)
fmt.Println("sex:", *sex)

打包

go build -o flag

如果不指定参数,则使用默认值

./flag

输出结果

name: 张三
age: 18
sex: true

指定参数

./flag --name 李四 --age 20 --sex 1
name: 张三
age: 18
sex: true

结果还是默认值,这是为什么呢?原因就是没有进行解析.

name := flag.String("name", "张三", "姓名")
age := flag.Int("age", 18, "年龄")
sex := flag.Bool("sex", true, "性别")
flag.Parse()
fmt.Println("name:", *name)
fmt.Println("age:", *age)
fmt.Println("sex:", *sex)

flag.Parse()表示对命令行参数进行解析并且绑定。

再次尝试

name: 李四
age: 20
sex: true
2. 根据指针类型获取
// 格式
// flag.StringVar(要绑定的变量指针, 参数名, 默认值, 参数描述)
func main() {
	var name string
	var age int
	var isN bool
	flag.StringVar(&name, "name", "张三", "名称")
	flag.IntVar(&age, "age", 18, "年龄")
	flag.BoolVar(&isN, "sex", true, "性别")

	flag.Parse()
	fmt.Println("name:", name)
	fmt.Println("age:", age)
	fmt.Println("sex:", isN)
}
go build -o flag

根据-h命令 查看参数描述

./flag -h
Usage of ./flag:
  -age int
        年龄 (default 18)
  -name string
        名称 (default "张三")
  -sex
        性别 (default true)
3. 参数携带方式

如flag.StringVar(&name, "name", "张三", "名称")

方式有如下几种

  • -name 张三
  • –name 张三
  • -name=张三
  • –name=张三
4. 命令行合法参数值

flag包支持的命令行参数类型有bool、int、int64、uint、uint64、float float64、string、duration。

flag参数 有效值
字符串flag 合法字符串
整数flag 1234、0664、0x1234等类型，也可以是负数。
浮点数flag 合法浮点数
bool类型flag 1, 0, t, f, T, F, true, false, TRUE, FALSE, True, False。
时间段flag 任何合法的时间段字符串。如”300ms”、”-1.5h”、”2h45m”。 合法的单位有”ns”、”us” /“µs”、”ms”、”s”、”m”、”h”。
