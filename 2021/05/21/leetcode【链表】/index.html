1.链表节点删除

给定单向链表的头指针和一个要删除的节点的值，定义一个函数删除该节点。返回删除后的链表的头节点

Leetcode: https://leetcode-cn.com/problems/shan-chu-lian-biao-de-jie-dian-lcof

Code

public static ListNode deleteNode(ListNode head, int val) {
    /** 如果头结点就是要删除的节点*/
    if (head.val == val){
        head = head.next;
        return head;
    }
    /** 临时节点存储*/
    ListNode temp = head;
    ListNode prevNode = temp;
    do {
        if (val == temp.val) {
            break;
        }
        prevNode = temp;
        temp = temp.next;
    } while (null != temp);
    /** 如果找到了要删除的节点*/
    if (prevNode.next.val == val) {
        if (null == prevNode.next.next) {
            prevNode.next = null;
        } else {
            prevNode.next = prevNode.next.next;
        }
    }
    return head;
}

Test

public static void main(String[] args) {
    ListNode header = new ListNode(4);
    ListNode node1 = new ListNode(5);
    ListNode node2 = new ListNode(1);
    ListNode node3 = new ListNode(9);
    header.next = node1;
    node1.next = node2;
    node2.next = node3;
    System.out.println(header);
    header = deleteNode(header,9);
    System.out.println(header);
}

2.链表反转

给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

Leetcode: https://leetcode-cn.com/problems/reverse-linked-list/

Code

递归实现

/** 递归实现*/
public ListNode reverseList(ListNode head) {
    if (head == null || head.next == null) return head;
    ListNode newHead = reverseList(head.next);
    head.next.next = head;
    head.next = null;
    return newHead;
}

非递归实现

/** 非递归实现*/
public ListNode reverseList(ListNode head) {
    if (head == null || head.next == null) return head;
    ListNode newHead = null;
    while (head != null){
        ListNode temp = head.next;
        head.next = newHead;
        newHead = head;
        head = temp;
    }
    return newHead;
}

3.判断一个链表是否有环

给定一个链表，判断链表中是否有环。如果链表中存在环，则返回 true 。 否则，返回 false 。

Leetcode: https://leetcode-cn.com/problems/linked-list-cycle/

快慢指针实现,两个指针去遍历,每个指针的步长不同 ,如果重叠就表示是环形链表

Code

public boolean hasCycle(ListNode head) {
    if (head == null || head.next == null) return false;
    ListNode slow = head;//慢指针
    ListNode fast = head.next;//快指针
    while (fast != null && fast.next != null){
        if (slow == fast) return true; //一旦两个指针重叠,就是环形链表
        slow = slow.next;
        fast = fast.next.next;
    }
    return false;
}
