Vue3学习之旅【四】Vuex

发表于2021-05-04|更新于2021-05-07|Vue系列
|字数总计:511|阅读时长:2分钟|阅读量:

1. 什么是Vuex?

一个全局的数据管理工具,就是这么简单,主要实现便于组件之间的数据通信！

2. Vuex安装

  • npm

    npm install vuex --save

  • yarn

    yarn add vuex

3. 概念

  • State

    全局状态(数据)存储区域

  • Getters

    类似于组件中的计算属性

  • Mutations

    操作state数据的函数

  • Actions

    异步操作

  • Modules

4. 基本使用

  • 在Vuex中 定义公共数据
import { createStore } from 'vuex'

export default createStore({

  state: {/** 全局状态(数据)存储区域*/
    count: 0
  },
  mutations: { /** 操作state数据的函数*/
    incr(state){
      state.count++
    },
    decr(state){
      state.count--
    },
    /** 带参数的操作函数*/
    addByNum(state,num){
      state.count += num
    },
    /** 带对象参数的操作函数*/
    setObj(state,obj){
      state.obj = obj
    }
  },
  actions: {
        //调用后端API
    getToken(context){
      setTimeout(()=>{
        //调用mutations中的方法
        context.commit('getToken')
      },3000)
    }
  },
  modules: {
  }
})

  • 注册到Vue中

    import store from './store'
createApp(App).use(store).mount('#app')

  • 在任意组件中使用

    <script>
export default {
  name:'About',
  methods:{
    incr(){
      //派发Vuex中 mutations定义的操作函数
      this.$store.commit('incr')
    },
    decr(){
      this.$store.commit('decr')
    },
        /** 派发带有参数的函数*/
    addByNum(num){
      this.$store.commit('addByNum',num)
    },
    /** 派发带有对象参数的函数*/
    setObj(){
      this.$store.commit('setObj',{id : 1001,name: '小张'})
    },
     dispAsync(){
      this.$store.dispatch('getToken')
    }
  }
}

5.Vuex State模块化

  • 按业务划分,如User模块数据、Product模块数据

    //定义User模块State
const user = {
  state:()=>({
    userInfo:{
      name: 'lxz',
      age: 18
    }
  }),
  mutations:{},
  getters:{
    /** 模块化的计算属性,可选三个参数
     * 1.本身的state数据
     * 2.getters实例
     * 3.全局的state数据
     * */
    getUserInfo(userState,getters,rootState){
      return userState.userInfo
    },
    getCommonData(userState,getters,rootState){
      return rootState.obj
    }
  },
  actions:{}
}

export default createStore({

  state: {/** 全局状态(数据)存储区域*/
  },
  mutations: { /** 操作state数据的函数*/
    getName(state){
      console.log(state.names)
      return state.names
    }
  },
  getters:{/*Vuex的计算属性*/
  },
  actions: {/** 异步操作区域*/
    getInfo(context){
      context.commit('getName','params')
    }
  },
  modules: {/** 将状态数据进行模块化*/
    user //将User模块数据 注入到全局模块
  }
})

  • 使用模块化后的数据

    <h3>Vuex 模块化</h3>
User模块下的数据:{{$store.state.user.userInfo}}<br/>
调用User模块下的计算属性:{{$store.getters.getUserInfo}}<br/>
调用全局的数据{{$store.getters.getCommonData}}
