Go【三】字符处理

发表于2021-07-28|更新于2023-03-01|Go
|字数总计:2.3k|阅读时长:9分钟|阅读量:

定义一个简单字符串

str := "Hi满城雪"

1. 字符串长度

fmt.Println(len(str)) //9

但是我们发现,其实通过len获取的只是这个字符串的所占字节数,并非字符数

我们需要通过utf8包的RuneCountInString函数来获取字符数

fmt.Println(utf8.RuneCountInString(str)) //5

2. 遍历字符串

for index := range str {
  fmt.Println(index)
}
0 
1 
2 
5 
8

通过range遍历的字符串,index其实只是所占字节位,并非是具体的字符索引

接下来我们将字符串转为[]byte再遍历

for _, b := range []byte(str) {
  fmt.Printf("%X  ",b)
}
48  69  E6  BB  A1  E5  9F  8E  E9  9B  AA  //UTF8编码

可以看到,48与69分别是字符H与i,而其他中文字符则各占了三个字节,并且是以UTF8的格式编码

接下来再次遍历

for i, ch := range str {
  fmt.Printf("(%d,%X)",i,ch)
}
(0,48)(1,69)(2,6EE1)(5,57CE)(8,96EA) //Unicode编码,这是由上面的UTF8解码而来

Go的字符串遍历时,会对字符串进行一个UTF-8的解码,解出来的每一个字符又会转义为UniCode,将转义后的UniCode码,存储在这个类型为int32的ch中,这个ch其实就是一个rune类型

通过rune进行字符串的遍历

for index,ch := range []rune(str){
  fmt.Printf("%d %c \n",index,ch)
}
0 H 
1 i 
2 满 
3 城 
4 雪

将字符串转为rune后,index就会是每一个字符的索引,而ch还是每个字符串对应的Unicode码,所以需要通过%c来进行转义.

3. 字符串拼接

  1. 通过+标识符进行拼接
name1 := "张三"
name2 := "李四"
name3 := name1 + name2
fmt.Println(name3)

fmt.Println("Hello" + "World")

在Go中string是不可变的,每次对字符串的修改都会产生新的字符串,所以这种方式效率较低。

  1. 通过string.Join()函数拼接,这种方式需要构建slice。
name1 := "张三"
name2 := "李四"
name3 := strings.Join([]string{name1, name2}, "")
// 使用-作为连接符
name3 := strings.Join([]string{name1, name2}, "-")
fmt.Println(name3)
  1. 使用bytes.Buffer,更高效的方式。
buffer := bytes.Buffer{}
buffer.WriteString("Hello")
buffer.Write([]byte("今天天气真不错!"))
fmt.Println(buffer.String()) //Hello今天天气真不错!

4. 字符串比较

  1. strings.Compare(s1,s2) int 返回一个整数,按照字典前后顺序比较两个字符串。
fmt.Println(strings.Compare("a", "a")) //0
fmt.Println(strings.Compare("c", "b")) //1
fmt.Println(strings.Compare("a", "b")) //-1
  1. strings.EqualFold(s1,s2) bool比较两个字符的值,忽略大小写
fmt.Println(strings.EqualFold("Hello", "hello")) //true
fmt.Println(strings.EqualFold("Hello", "World")) //false
  1. 使用==也可以比较,这种方式会区分大小写
fmt.Println("hello" == "hello") //true
fmt.Println("hello" == "Hello") //false

5. 字符串截取

不包含中文的字符截取

如果不包含中文,可以将字符串转为[]byte,然后直接对[]byte进行切片截取。如下 截取文本中索引为1到索引3的字符。切片的截取规则是 包含起始索引元素,但不包含截止索引元素

text := "abcdef"
text = string([]byte(text)[1:3])
fmt.Println(text)//bc

包含中文的字符截取

如果字符包含中文,需要转为[]rune切片,rune类型中,一个字符或一个中文都占一位索引,然后对该切片截取即可。

body := "hi,今天天气真好!"
body = string([]rune(body)[3:9])
fmt.Println(body) //今天天气真好

6. 字符串替换

//int参数为最多替换多少个复合条件的字符,如果为 -1 表示替换全部
text := strings.Replace("ABAACCAAE", "A", "a", 3)
fmt.Println(text) //aBaaCCAAE

text := strings.Replace("ABAACCAAE", "A", "a", -1)
fmt.Println(text) //aBaaCCaaE

//该方法= strings.Replace(text, oldchart, newchat, -1)
text = strings.ReplaceAll("ABAACCAAE","A","a")
fmt.Println(text) //aBaaCCaaE

7. 字符大小写转换

小写转大写

fmt.Println(strings.ToUpper("abcD")) //ABCD

大写转小写

fmt.Println(strings.ToLower("ABDC")) //abdc

8. 字符是否包含子串

strings.Contains(s1,s2) bool 查询一个字符串是否包含另一个字符串。区分大小写。

fmt.Println(strings.Contains("Hello", "H")) //true
fmt.Println(strings.Contains("Hello", "h")) //false

strings.ContainsAny(s1, s2) bool 查询一个字符串是否包含另一个字符串任意字符。

fmt.Println(strings.ContainsAny("Hello", "o"))  //true
fmt.Println(strings.ContainsAny("Hello", "a"))  //false
fmt.Println(strings.ContainsAny("Hello", "oa")) //true

strings.ContainsRune(s1, rune)查询字符串中是否包含指定的rune。

fmt.Println(strings.ContainsRune("今天天气真不错!", '天')) //true
fmt.Println(strings.ContainsRune("今天天气真不错!", '气')) //true

9. 统计子串出现次数

strings.Count(s1,s2)int统计一个字符在一个字符串中出现的次数。如果s2为空字符串,则返回s1的(字符数量 +1)

fmt.Println(strings.Count("我也想过过过儿过过的生活", "过")) //5
//如果s2为空字符,返回的是 utf8.RuneCountinStrings(s1) + 1
fmt.Println(strings.Count("我也想过过过儿过过的生活", "")) //12 + 1

10. 查询字符串

strings.Index(s1,s2)int 返回s2在s1中首次出现的索引,如果不存在则返回-1。

fmt.Println(strings.Index("Hello,World!", "r"))    //8
fmt.Println(strings.Index("Hrello,World!rrr", "r")) //1

strings.LastIndex(s1,s2)int 返回最后一次出现的索引,不存在返回-1，如果s2为空字符串则返回s1的长度。

fmt.Println(strings.LastIndex("Hello,World!", "r"))    //8
fmt.Println(strings.LastIndex("Hello,World!rrr", "r")) //14
fmt.Println(strings.LastIndex("Hello,World!rrr", "")) //15
// 在 s 中查找 sep 的第一次出现，返回第一次出现的索引
func Index(s, sep string) int
// 在 s 中查找字节 c 的第一次出现，返回第一次出现的索引
func IndexByte(s string, c byte) int
// chars 中任何一个 Unicode 代码点在 s 中首次出现的位置
func IndexAny(s, chars string) int
// 查找字符 c 在 s 中第一次出现的位置，其中 c 满足 f(c) 返回 true
func IndexFunc(s string, f func(rune) bool) int
// Unicode 代码点 r 在 s 中第一次出现的位置
func IndexRune(s string, r rune) int

// 有三个对应的查找最后一次出现的位置
func LastIndex(s, sep string) int
func LastIndexByte(s string, c byte) int
func LastIndexAny(s, chars string) int
func LastIndexFunc(s string, f func(rune) bool) int

获取字符串中文所在位置

获取首个中文字符所在字节索引。

//过滤函数: 判断单个rune 是否为中文汉字
filter := func(r rune) bool {
  return unicode.Is(unicode.Han, r)
}
//获取首个中文位置,返回-1 表示不包含中文
index := strings.IndexFunc("HelloWorld啊", filter)
fmt.Println(index) //10

获取最末尾个中文字符所在字节索引。

index = strings.LastIndexFunc("HelloWorld啊哈", filter)
fmt.Println(index) //13

11. 字符串分割

  1. strings.Fields(s) ：根据空格进行字符串切割,返回字符串切片,如果字符串只包含空格,则返回空切片。

    fmt.Println(strings.Fields("hi zero say")) //[hi zero say]
fmt.Println(strings.Fields("     "))       //[]

  2. strings.Split(s1,sep):根据指定的字符进行分割

    fmt.Println(strings.Split("hi,今天天气,真不错!", ","))//[hi 今天天气 真不错!]

    该方式等同于strings.SplitN(s1,sep,-1)

  3. strings.SplitAfter(s,sep): 根据指定的字符分割,并且保留分隔符。

    fmt.Println(strings.SplitAfter("hi,今天天气,真不错!", ",")) //[hi, 今天天气, 真不错!]
fmt.Println(strings.SplitAfterN("hi,今天天气,真不错!", ",", -1)) //[hi, 今天天气, 真不错!]

12. 字符前后缀匹配

  1. strings.HasPrefix(s,prefix)s的前缀是否为prefix。

    fmt.Println(strings.HasPrefix("Hello", "h")) //false
fmt.Println(strings.HasPrefix("Hello", "H")) //true

  2. strings.HasSuffix(s,suffix)s的后缀是否为suffix。

    fmt.Println(strings.HasSuffix("Hello", "o")) //true
fmt.Println(strings.HasSuffix("Hello", "O")) //false

13. 字符串叠加

fmt.Println(strings.Repeat("哈", 4)) //哈哈哈哈

14. 字符串映射处理

通过strings.Map(func,str)能够实现以上很多功能,该函数就是针对每个rune字符都进行一遍过滤,其过滤逻辑由我们自定义的函数为准。如下所示

mapping := func(r rune) rune {
	switch {
	//如果是大写字符,就过转为小写
	case unicode.IsUpper(r):
		return unicode.ToLower(r)
	//如果为小写字符,则不做处理,直接返回
	case unicode.IsLower(r):
		return r
	//如果为中文字符,则转为换行符
	case unicode.Is(unicode.Han, r):
		return '\n'
	}
	//其他任何非字母、汉子的字符全部排除掉
	// 返回-1 表示抛弃掉该rune
	return -1
}
str := strings.Map(mapping, "Hello你#￥%……\n（'World\n,好Hello^(&(*界gopher...")
fmt.Println(str)

15. 字符串修剪

strings.Trim(s,cutset)

//去除字符前后的空格
fmt.Println(strings.Trim(" hi,你好  ", " ")) //hi,你好
//如果字符前后缀与要清除的字符一致,则清除两端,否则不作处理
fmt.Println(strings.Trim("为所欲为", "为"))   //所欲
fmt.Println(strings.Trim("s为所欲为s", "为")) //s为所欲为s

//只清除左侧或右侧
strings.TrimLeft()
strings.TrimRight()
// 将 s 左侧和右侧中匹配 cutset 中的任一字符的字符去掉
func Trim(s string, cutset string) string

func TrimLeft(s string, cutset string) string

func TrimRight(s string, cutset string) string
// 如果 s 的前缀为 prefix 则返回去掉前缀后的 string , 否则 s 没有变化。
func TrimPrefix(s, prefix string) string
// 如果 s 的后缀为 suffix 则返回去掉后缀后的 string , 否则 s 没有变化。
func TrimSuffix(s, suffix string) string

func TrimSpace(s string) string

func TrimFunc(s string, f func(rune) bool) string
